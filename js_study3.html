JavaScript - 현재 날짜, 시간 가져오기
(년, 월, 일, 시, 분, 초 구하기)

현재(now) 날짜, 시간 구하기 - new Date()

2021년 7월 12일: new Date("2021-07-12")

연도 구하기 - getFullYear()

new Date에서 getFullYear()로 연도만 가져온다.

var day = new Date().getFullYear();

월 구하기 - getMonth()

new Date에서 getMonth()로 월을 가져온다.

원래 월 -1 의 값으로 나온다.(즉, 1~12월이 0~11의 수로) 그래서 월을 구할때는 getMonth 값에 +1을 주어야함

var month = new Date().getMonth() + 1;

일자 구하기 - getDate()

new Date에서 getDate()로 일자를 가져온다

var date = new Date().getDate();

요일 구하기 - getDay()

new Date에서 getDay()로 요일을 가져온다.

var day = new Date().getDay();

일요일은 0, 월요일은 1, 화요일은 2, 수요일은 3, 목요일은 4, 금요일은 5, 토요일은 6 을 반환한다.

분 구하기 - getMinute()

new Date에서 getMinute()로 분을 가져온다.

var minute = new Date().getMinute();

초 구하기 - getSeconds()

new Date에서 getSeconds()로 초을 가져온다.

var seconds = new Date().getSeconds();
