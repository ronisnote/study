JavaScript -객체 (Object)

객체 = 여러 개의 자료형을 한번에 저장하는 자료형

배열과 객체는 상당히 유사 (배열을 객체를 기반으로 만들었기때문에)

다른점은 배열은 요소에 접근할 때 인덱스를 사용하고 객체는 속성에 접근할 때 키를 사용합니다.

객체의 선언

* print와 같이 객체의 속성중 자료형이 함수인 속성을 메소드라고 한다.

객체를 반복문으로 불러오기

for in 문

배열과 객체를 사용하여 여러 개의 데이터

생성자 함수 = '객체를 만드는 함수'

* 일반적인 함수와 구분하기 위해 생성자 함수는 대문자로 시작

함수를 생성하고 new 키워드를 사용하여 객체를 생성

프로토 타입을 이용한 메소드 생성 & 객체 배열

생성자 함수로 만든 객체는 프로토타입이라는 공간에 메소드를 지정해서 모든 객체가 공유하도록 한다.

* 객체 지형적으로 구성한 객체 배열
